# Big T BBQ Demo Site

Modern, mobile-first Next.js + Tailwind template for **Big T BBQ** with a "Modern Rustic + Street" vibe.

## Run locally

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Project structure

- `app/layout.tsx` - metadata and Google fonts
- `app/page.tsx` - single-page landing with all sections
- `app/globals.css` - theme variables + global styles
- `public/brand/logo.png` - brand logo
- `public/food/` - hero + signature plate images
